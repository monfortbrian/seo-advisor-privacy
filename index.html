Privacy Policy

SEO Advisor - Chrome Extension
Last updated: 2026-02-17

SEO Advisor (“the extension”) is designed to help ecommerce store owners and marketers identify and fix on-page SEO issues. This Privacy Policy explains what data is processed, how it is used, and how it is protected.

1. Overview

SEO Advisor analyzes ecommerce product pages only when a user explicitly initiates a scan. The extension reads visible on-page SEO elements (such as metadata, headings, images, and structured data) to generate guidance. No background monitoring or passive browsing tracking occurs.

2. Information Processed

a) Page Content (Local Analysis)

When a scan is triggered, the extension temporarily reads the active webpage’s SEO-relevant content to perform analysis. This includes elements such as:

  • Meta tags and page titles
  • Headings and text structure
  • Images and alt attributes
  • Structured data (schema markup)

This content is processed only for SEO analysis and guidance generation.

b) Account Information

If the user signs in, authentication is handled via Google OAuth and Supabase. Basic account data such as email address and subscription status may be stored securely to manage usage limits and plan features.

c) Local Storage

The extension stores limited data locally in the browser, including:

  • Scan usage counts and plan limits
  • Recent analysis results
  • User interface preferences

This data improves performance and user experience and can be removed by uninstalling the extension.

3. How Data Is Used

Data is used solely to:

  • Analyze product pages for SEO issues
  • Provide step-by-step SEO guidance
  • Manage subscription features and scan limits
  • Display analysis results in the extension interface

4. Data Sharing and Sale

SEO Advisor does not sell, rent, or trade user data. Data is not shared with third parties except for essential infrastructure providers required to operate the service (e.g., authentication and database services).

5. No Tracking or Advertising

The extension does not use trackers, advertising identifiers, or cross-site monitoring. Browsing activity outside user-initiated scans is not collected or observed.

6. Data Retention

Local analysis data remains in the user’s browser until cleared or the extension is removed. Account data is retained only as long as the user maintains an active account or subscription.

7. Security

Account and subscription data are stored using Supabase secure infrastructure. Authentication is handled through Google OAuth. Reasonable technical safeguards are applied to protect stored data.

8. User Control

Users can:

  • Use the extension without signing in (limited features)
  • Clear local data by uninstalling the extension
  • Request account deletion by contacting support

9. Children’s Privacy

SEO Advisor is intended for business and professional use and is not directed to children under 13.

10. Changes

This Privacy Policy may be updated to reflect product or regulatory changes. Updates will be posted on this page with a revised date.
